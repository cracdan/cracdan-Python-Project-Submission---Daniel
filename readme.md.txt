# IMDb Top Movies Scraper

A Python script that scrapes IMDb's Top 250 movies list, extracts movie titles and their genres, and provides a simple movie recommendation system based on genres.

## Features

- Scrapes IMDb's Top 250 movies list
- Extracts movie titles and their associated genres
- Saves data to CSV format
- Provides movie recommendations based on genre input
